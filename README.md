# Money Management

A simple money management web app using HTML, CSS, and JavaScript.

## Getting Started

No build step is required. Simply open `index.html` in your browser.

## Installation (Optional)
If you want to manage dependencies or use npm scripts, run:

```bash
npm install
```

## Project Structure
- `index.html`: Main HTML file
- `styles.css`: CSS styles
- `app.js`: JavaScript logic
- `package.json`: Node.js metadata (optional)
- `requirements.txt`: (empty, for Python projects only)
